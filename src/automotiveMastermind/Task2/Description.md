## Convert Arabic numerals to Roman one 

Write a function that accepts an integer from 1-1000 and returns a string that is the roman numeral representation of the provided input.

For reference:

|Symbol|Value|
|:----:|:---:|
| I    | 1   |
| V    | 5   |
| X    | 10  |
| L    | 50  |
| C    | 100 |
| D    | 500 |
| M    | 1000|
